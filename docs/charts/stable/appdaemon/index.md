# appdaemon

![Version: 11.0.7](https://img.shields.io/badge/Version-11.0.7-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: "4.2.1"](https://img.shields.io/badge/AppVersion-"4.2.1"-informational?style=flat-square)

AppDaemon is a loosely coupled, multithreaded, sandboxed python execution environment for writing automation apps for Home Assistant home automation software. It also provides a configurable dashboard (HADashboard) suitable for wall mounted tablets.
